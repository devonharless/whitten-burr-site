Offerings

We carry projects from Discovery and Ideation through Design and Development.

We analyze the project’s Requirements, create a Project Charter and a Sitemap, and set up a plan to execute the project.

We create Wireframes and User Flows that help layout the project. We mock-up Design Comps and other visual elements that show how the site will look when it is finally live.

We build project using a toolkit of HTML5, CSS3, and JavsScript. We test features and interactivity of working prototypes and debug before launch.
